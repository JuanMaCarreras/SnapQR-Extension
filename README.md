# SnapQR
> Chrome extension that lets you scan any QR code directly from your screen — no phone needed.

## How it works
Click the extension icon, draw a rectangle over any QR code on your screen, and instantly get the link. No popups, no extra steps.

## Features
- 🖱️ Select any region of the screen to scan
- 🔗 Get the URL in a clean popup with one click to open
- 📋 Copy the link to clipboard
- ⚡ Works directly on the page, no interruptions

## Built with
- Chrome Extensions API (Manifest V3)
- [jsQR](https://github.com/cozmo/jsQR) — QR decoding library
- Vanilla JavaScript

## Installation
1. Clone this repo
2. Go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the project folder

# SnapQR

> Extensión de Chrome que te permite escanear cualquier código QR directamente desde tu pantalla — sin necesidad de usar el móvil.

## Cómo funciona
Haz clic en el icono de la extensión, dibuja un rectángulo sobre cualquier código QR en tu pantalla y obtén el enlace al instante. Sin ventanas emergentes, sin pasos extra.

## Funcionalidades
- 🖱️ Selecciona cualquier región de la pantalla para escanear
- 🔗 Obtén la URL en un popup limpio con un clic para abrirla
- 📋 Copia el enlace al portapapeles
- ⚡ Funciona directamente sobre la página, sin interrupciones

## Tecnologías
- Chrome Extensions API (Manifest V3)
- [jsQR](https://github.com/cozmo/jsQR) — librería de decodificación QR
- JavaScript puro

## Instalación
1. Clona este repositorio
2. Ve a `chrome://extensions`
3. Activa el **Modo desarrollador**
4. Haz clic en **Cargar descomprimida** y selecciona la carpeta del proyecto
